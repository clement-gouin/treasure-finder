# Treasure Finder
*Maybe the real treasure is the friends we make along the way *

### [Tool link](https://clement-gouin.github.io/treasure-finder/)

## Data format

Format is made line by line, a point is 4 lines 

Header (3 lines):
```txt
1   latitude
2   longitude
3   secret (html)
4   point name
```

## Sample

```txt
48.85832397934772
2.2940806701383734
<i>You found me !</i>
sample point
```
